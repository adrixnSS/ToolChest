# ToolChest.site 🧰
### La Suite Maestra de Archivos: Privada, Rápida y Local.

**ToolChest** es una aplicación web de herramientas multiuso diseñada para procesar vídeo, imágenes y documentos directamente en tu navegador. Con un enfoque radical en el cumplimiento legal (RGPD) y la experiencia de usuario (UX) premium, ToolChest combina la potencia de motores locales con una interfaz editorial elegante.

## ✨ Características Principales

### 🎬 Estudio de Vídeo
*   **Recorte de Precisión**: Línea de tiempo interactiva con arrastre (Drag & Drop) y botones de ajuste fino de +/- 1 segundo.
*   **Sincronización de Frames**: Visualiza el frame exacto de inicio y fin mientras ajustas el recorte en tiempo real.
*   **Conversión Multipropósito**: Exporta a MP4, MKV, MOV, AVI o genera GIFs de alta calidad.
*   **Optimización**: Compresión inteligente para reducir el peso de tus archivos sin perder calidad profesional.

### 🖼️ Laboratorio de Imagen
*   **Borrador de Fondos Mágico**: Elimina fondos de forma automática y local, sin subir tus fotos a servidores externos.
*   **Redimensión Inteligente**: Cambia el tamaño y formato (PNG, JPG, WEBP) con un solo clic.
*   **Privacidad Total**: Todo el procesado de imagen ocurre al 100% en tu dispositivo.

### 📄 Editor de PDF
*   **Firma Digital Profesional**: Firma tus documentos PDF con una interfaz de dibujo fluida y sincronizada.
*   **Gestión de Metadatos**: Edita título, autor, asunto y palabras clave. El sistema precarga automáticamente los datos existentes del archivo.
*   **Optimizador de Lectura**: Renderizado de páginas con ajuste automático al ancho de pantalla ("Fit-to-Width").

## 🛠️ Tecnología y Privacidad
*   **Motores Locales**: Uso de `pdf.js` y `pdf-lib` para manipulación de documentos en el cliente.
*   **Seguridad**: Priorizamos el procesado *Client-Side* para garantizar que los archivos sensibles nunca abandonen el equipo del usuario cuando sea posible.
*   **RGPD / GDPR Ready**: Incluye banner de consentimiento (Consent Mode v2) y páginas legales completas (Privacidad, Cookies, Aviso Legal).

## 🚀 Instalación y Despliegue

### Requisitos previos
*   **Node.js** (v14 o superior)
*   **FFmpeg** instalado en el sistema (necesario para el procesamiento de vídeo en el servidor)

### Pasos para ejecución local
1. Clona el repositorio:
   ```bash
   git clone https://github.com/adrixnSS/file-converter-pro.git
   cd file-converter-pro
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm start
   ```
4. Abre `http://localhost:3000` en tu navegador.
